About Me

Background

Giga Factory is a website I created for selling Giga Cannon(An Online game item), which helps many players find the items and its components for a better price.

My Role

• Web design

• UX / UI

• Coding (HTML, CSS, JS)

Year

2021

Problem

The item is difficult to make

The Giga Cannon is one of the most expensive and most wanted items in the 12Tails online game. The first one was sold for real money, for around 5,000 baht. However, to make a Giga Cannon, players must collect numerous components and assemble them, which is a laborious and time-consuming process.

Solution

Create a marketplace for the items

The plan is to set up a website dedicated to purchasing all of the Giga Cannon components, assembling them into Giga Cannons, and selling them for a profit.

Test + Improvement

Simplify the website

When I asked my friends for feedback on the first prototype, the majority of them said that the website was too complicated and overwhelming because there were so many component listings. So I changed the listing to a table format similar to the trading website. The website is now cleaner and easier to navigate.

Final Product

Result

6,000 baht in profit.

The website was popular in the first week after launch. Unfortunately, due to increased supply, the price of Giga cannons is steadily declining. The website was shut down in late 2021, but it still generated a profit of about 6,000 baht in total.
